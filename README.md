[![NPM downloads](http://img.shields.io/npm/dm/umi-plugin-extract-auth.svg?style=flat)](https://npmjs.org/package/umi-plugin-extract-auth)



## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['umi-plugin-umi-plugin-extract-auth', options],
  ],
}
```
